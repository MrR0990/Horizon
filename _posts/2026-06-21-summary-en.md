## [Android 17 Introduces Per-App Memory Limits](https://www.reddit.com/r/androiddev/comments/1ubgnw0/android_app_memory_limits_in_place_from_android_17/) ⭐️ 8.0/10

Starting from Android 17, the platform adds per-app memory limits alongside the existing system-wide Low Memory Killer Daemon (LMKD), allowing the system to kill an individual app that exceeds its allocated memory budget even when the device has sufficient overall RAM. This change fundamentally alters how Android apps manage memory, requiring developers to optimize memory usage more strictly to avoid unexpected termination, and it shifts the memory management model from reactive (system-wide pressure) to predictive (per-app caps). The per-app memory ceiling is based on the device's total RAM and is intentionally conservative in the first release; apps that exceed the limit are terminated with a specific exit reason, making it easier for developers to diagnose memory issues.

reddit · r/androiddev · /u/meonlineoct2014 · Jun 21, 04:33

**Background**: Before Android 17, Android relied solely on the Low Memory Killer Daemon (LMKD) to kill the least important processes when the system ran low on memory, based on process priority (e.g., foreground vs. background). The new per-app limits add a second mechanism that can kill a memory-hungry app even when the device is not under global memory pressure, encouraging better memory hygiene.

## [WeChat Tests Native AI Assistant 'Xiao Wei' in Small Beta](https://36kr.com/newsflashes/3862458180359424?f=rss) ⭐️ 6.0/10

WeChat has begun a small-scale beta test of its native AI assistant 'Xiao Wei', accessible via an eye-shaped icon on the main interface. The assistant supports text and voice commands to perform core WeChat functions and invoke mini-programs. This marks WeChat's first native AI assistant integration, potentially transforming how users interact with the app. It could pave the way for deeper AI features within China's largest messaging platform, affecting billions of users. The beta is limited to a small group of users, and the assistant can help with tasks like sending messages to friends, querying Moments, and booking services. It is unclear which underlying AI model powers Xiao Wei, though WeChat has previously tested DeepSeek-R1 for AI search.
